Documentation Page

The web application, My Cinema, will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Decription URL HTTP Method Request body data format. Response body data format.
Return a list of all movies to the user. /movies get None JSON object holding data about all movies.
Return data about a specific movie by title to the user. /movies/[title] get None JSON object containing the movies title, director, description, genre, and image URL.
Return data about a genre by name. /movies/genre/[name] get None JSON object holding data about the specific genre.
Return data about a director by name /movies/director/[name] get None JSON object holding data about the director including bio, birth year, and death year.
Allow new users to register. /users post JSON object holding information about the user needing to be added. JSON object holding information about the user added.
Allow users to update their user info. /users/[username] put None A text showing username has been updated.
Allow users to add a movie to their list of favorites. /users/[username]/[title] post None A text that a movie has been added
Allow users to remove a movie from their list of favorites. /users/[username]/[title] delete None A text that a movie has been removed
Allow existing users to deregister. /users/[username] delete None A text that a user email has been removed.
