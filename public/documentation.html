Documentation Page

The web application, My Cinema, will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Decription URL HTTP Method Request body data format. Response body data format.
Return a list of all movies to the user. /movies get None JSON object holding data about all movies.
Return data about a specific movie by title to the user. /movies/[title] get None JSON object containing the movies title, director, description, genre, and image URL.
Return data about a genre by name. /movies/genre/[name] get None JSON object holding data about the specific genre.
Return data about a director by name. /directors/[name] get None JSON object holding data about the director including bio, birth year, and death year.
Allow new users to register. /users post JSON object holding information about the user needing to be added. Example
{
Username: HellyZelly, Password: Heather123!, Email: HellyZelly@gmail.com, Birthday: 1961-01-03 }		 JSON object holding information about the user added. Structured like:
{
ID: 1, Username: HellyZelly, Password: Heather123, Email: HellyZelly@gmail.com, Birthday: 1961-01-03 }
Allow users to update their user info. /users/[username] put A JSON object holding data to update. A JSON object holding the updated user.
Allow users to add a movie to their list of favorites. /users/[username]/movies/[MovieID] post None A JSON object holding the updated user.
Allow users to remove a movie from their list of favorites. /users/[username]/movies/[MovieID] delete None A JSON object holding the updated user.
Allow existing users to deregister. /users/[username] delete None A text that a user has been deleted.
